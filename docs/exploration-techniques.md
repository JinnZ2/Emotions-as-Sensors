# Exploration Techniques for Non-Local Pattern Correlation

## Purpose

The **Non-Local Pattern Correlation Sensor** (Sensor Category 6: Advanced
Pattern Recognition) detects correlated patterns between systems that lack a
traceable direct information channel. This is the phenomenon the older
"morphic resonance" label was gesturing at, named here in mechanism-neutral
terms.

Rather than commit the framework to a single causal hypothesis, the sensor
exposes a **menu of grounded exploration techniques**. The investigator
selects the technique whose scope matches the question — same principle as
the framework's existing parallel-definition stance (Western biomedical /
relational / Indigenous side-by-side, no winner picked).

## Why a menu, not a default

- **Framework consistency.** Emotions-as-Sensors already preserves multiple
  definitions (Western biomedical, relational, Indigenous) without picking a
  winner. Hardcoding a single replacement for "morphic resonance" would
  violate the framework's own pluralism.
- **Different questions need different lenses.** Trans-generational trauma
  transmission is best investigated through epigenetics. Landscape-scale
  pattern persistence is best investigated through geological/ecological
  memory. Distant structural correlation is best investigated through
  metamaterial resonance principles. One sensor cannot fit all three without
  losing grounding in any of them.
- **Investigator sovereignty.** Lets the person doing the sensing choose the
  exploration technique appropriate to their tradition, training, and
  question. Same principle as sovereign nodes in kitchi — no central
  authority dictating the tool.

## The Techniques

Each technique below is a **lens**, not a ranking. They differ in scope, not
quality. Pick the one(s) whose scope matches the correlation you are
investigating.

### 1. Epigenetic Inheritance

- **When to use:** trans-generational patterns within biological lineages.
- **Grounded in:** methylation, histone modification, ncRNA inheritance
  (peer-reviewed).
- **Strength:** mainstream science; falsifiable; lab-reproducible.
- **Limitation:** biological systems only.

### 2. Mitochondrial Signaling

- **When to use:** somatic state persistence within an individual;
  intergenerational maternal transmission.
- **Grounded in:** retrograde signaling, mtDNA inheritance, metabolic
  memory.
- **Strength:** real biochemistry; explains how emotional/somatic state
  persists without requiring neurological encoding.
- **Limitation:** narrower temporal scope than epigenetics.

### 3. Gene-Expression Dynamics

- **When to use:** real-time response of biology to social/relational
  signals.
- **Grounded in:** social genomics (Cole et al., CTRA);
  transcriptomics.
- **Strength:** establishes the social-to-biological feedback loop
  directly.
- **Limitation:** individual-scale; doesn't address non-local correlation
  between separated systems.

### 4. Landscape Memory

- **When to use:** place-scale temporal continuity; land that carries
  pattern across centuries.
- **Grounded in:** geological recording, sediment archives, drainage
  patterns, ecological succession; aligned with Indigenous
  landscape-encoded knowledge.
- **Strength:** physical record + ongoing influence; bridges Western
  geology and traditional knowledge systems.
- **Limitation:** not informational in a quick-feedback sense; operates on
  decade-to-millennia scale.

### 5. Metamaterial Resonance Principle

- **When to use:** structurally similar systems showing correlated
  behaviors without direct channel.
- **Grounded in:** metamaterial physics, demonstrated in engineered
  systems.
- **Strength:** real mechanism for structure-determined collective
  behavior.
- **Limitation:** extends from engineered to natural systems by analogy;
  the analogy must be checked carefully each time.

### 6. Specific Quantum Effects

- **When to use:** documented quantum-biological phenomena
  (photosynthetic energy transfer, avian magnetoreception, FRET coupling).
- **Grounded in:** specific, replicated quantum-biology literature — *not*
  general "quantum consciousness" claims.
- **Strength:** scientifically sound where established.
- **Limitation:** high credibility cost if extended beyond demonstrated
  effects; treat as a narrow technique, not a general framework.
- **Caution:** if a quantum framing is being used where a classical
  mechanism would suffice, prefer the classical framing.

### 7. Field Correlation Observation

- **When to use:** when no current mechanism fits and the effect is
  reproducible across independent observations.
- **Grounded in:** the U(t) protocol — named uncertainty, not a parking
  lot.
- **Strength:** honest about the limit of current models.
- **Limitation:** requires the U(t) routing protocol to remain
  disciplined; without it, this technique becomes the same gravity well
  the older label was.

## Selection Protocol

When investigating a non-local pattern correlation:

1. **Identify the scale** at which the correlation operates — cellular,
   individual, generational, landscape, or cross-system.
2. **Select the technique whose scope matches that scale** (see the table
   below).
3. **If multiple techniques fit, run them in parallel.** Convergent
   results across independent techniques strengthen the finding.
4. **If no established technique fits and the effect reproduces, route to
   U(t)** via the Field Correlation Observation technique.

### Scale → Technique Map

| Scale of correlation                              | Primary technique(s)                                    |
| ------------------------------------------------- | ------------------------------------------------------- |
| Cellular / sub-cellular state persistence         | Mitochondrial signaling; specific quantum effects       |
| Individual real-time biology ↔ social signal      | Gene-expression dynamics                                |
| Generational (within biological lineage)          | Epigenetic inheritance; mitochondrial signaling         |
| Place / land / decade-to-millennia scale          | Landscape memory                                        |
| Structurally similar but separated systems        | Metamaterial resonance principle                        |
| Reproducible, no established mechanism fits       | Field correlation observation (routes to U(t))          |

## What This Patch Drops

- Sheldrake's specific causal claim (memory-in-fields).
- The loaded term "morphic."
- The false unity that hid three different scales of phenomenon under one
  label.

## What This Patch Preserves

- The detection target itself (non-local pattern correlation is real and
  worth sensing for).
- Multi-definition pluralism (the framework's core stance).
- Investigator sovereignty.
- Non-collapse into monoculture.

## What This Patch Adds

- Explicit mapping from question scale to technique.
- Explicit handling of the case where no technique fits (routes to U(t)).
- Alignment with landscape-encoded knowledge work across the broader
  ecosystem.
